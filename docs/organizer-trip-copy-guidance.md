# คำแนะการเขียนทริป (ผู้จัด — ไม่แสดงในแอป)

เอกสารนี้เก็บบริบทการเขียนชื่อทริป คำโปรย และภาพรวม **สำหรับทีมงาน / ผู้จัด** เท่านั้น ไม่ได้ฝังในหน้าฟอร์มสร้างทริป เพื่อไม่ให้ข้อความยาวรบกวนผู้ใช้และไม่บังคับโทนการเขียนต่อผู้จองบนหน้ารายการทริป

---

## หมวด «ชื่อและคำโปรย» (ฟอร์มสร้างทริป)

**แนวทางหมวด:** ดึงจากโพสต์หรือไอเดียของคุณ — บอกชัดว่าไปไหน กี่วัน แล้วใส่ข้อความที่ทำให้อยากไปด้วย

### ชื่อทริป

โครงที่อ่านแล้วจำง่าย: **จุดหมาย + ระยะเวลา + จุดขายสั้น ๆ** (รถ / ไกด์ / มื้อพิเศษ ฯลฯ) — ไม่ต้องยาว แต่ควรโดดเด่นในรายการ

ตัวอย่างโทน placeholder ที่เคยใช้อ้างอิง: เขาใหญ่ 2 วัน 1 คืน — แคมป์ริมน้ำ · รวมมื้อเย็น

### คำโปรยสั้น

**หนึ่งประโยคที่ชวนคลิก** — บอกใครเหมาะ หรือได้อะไรที่ไม่ซ้ำทัวร์ทั่วไป ไม่ต้องทวนชื่อทริปยาว ๆ

ตัวอย่างโทน: เดินป่าเช้า พักกลางธรรมชาติ — เหมาะทีมเล็กที่อยากชาร์จพลังแบบมีแอดเวนเจอร์เบา ๆ

---

## หมวด «เล่าเรื่องทริป» — ภาพรวมทริป

**แนวทางหมวด:** ขยายจากโพสต์หรือสคริปต์ของคุณ — เขียนเป็นคนจริง ยิ่งมีบรรยากาศและลำดับวัน ยิ่งน่าไป

**โครงย่อหน้าที่แนะนำ (ไม่บังคับในระบบ):**

1. ภาพรวมและอารมณ์ทริป  
2. วันต่อวันหรือกิจกรรมหลัก  
3. ใครเหมาะ / ระดับความเหนื่อย  
4. รวมอะไรในราคา — ไล่จากที่คนอ่านโพสต์แล้วอยากรู้ต่อ  

ตัวอย่างโครงข้อความใน textarea (อ้างอิง): วันแรกรวมตัวที่… กิจกรรมหลัก… / เหมาะกับคนที่… ระดับความเหนื่อย… / ราคานี้รวม… ไม่รวม…

---

## หมายเหตุ

- ข้อความในเอกสารนี้ไม่ปรากฏบนหน้าสาธารณะของผู้จอง  
- ฟอร์มสร้างทริป (โหมดกระชับ) ใช้เฉพาะหัวข้อฟิลด์และ placeholder สั้น ๆ เพื่อไม่รบสายตา
